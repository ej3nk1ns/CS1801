Conway's Game of Life
Liz Jenkins
January 2016

The program is called Game.java, and needs to be compiled with
Oracle java (type 'javafx' first, on teaching).  Then compile with
the option -d . to force the correct directory structure:

javac -d . Game.java

And run with the correct classpath:

java game.Game

There were meant to be parameters to specify the input file too...
